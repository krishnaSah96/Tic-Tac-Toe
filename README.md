# Tic Tac Toe 🎮

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Two-player gameplay
- Detects winner automatically
- Detects draw
- Reset game option
- Clean and responsive interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone this repository.
2. Open `index.html` in your web browser.
